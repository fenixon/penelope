# Proyecto Penélope

## Descripción

Penélope es un sitio web donde los usuarios estarán al tanto de los eventos que se realizan en la ciudad de Paysandú.

## Desarrollo

Penélope está siendo desarrollado sobre las siguientes tecnologías:

- [PHP 5.6+](http://php.net/)
- [Apache web server compatible con la versión de PHP](https://httpd.apache.org/)
- [MySQL 5.5+](https://getcomposer.org/)
- [Composer](https://getcomposer.org/)
- [Guzzle](http://docs.guzzlephp.org/)
- [Smarty](http://www.smarty.net/)
- [Bootstrap](http://getbootstrap.com/)
- [jQuery](http://jquery.com/)

## Instalación

1. Ubicar el directorio del proyecto en donde `Apache` pueda acceder y leer los archivos.
2. Crear una base de datos en `MySQL` y completar los datos respesctivos en el archivo `config/config.php`.
3. Ejecutar los scripts SQL en la base de datos deseada, **en orden**.
4. Ejecutar `composer` para instalar las dependencias de Penélope.
